These data files are provided to allow verification of a successful installation.

tcfa_in.sdf and tmpl.sdf are inputs for tcfa.
comfa_in.sdf is both tcfa's output and comfa's input.
model.tcfa and the .oesrf files are comfa's output.

Note that tcfa_in.sdf was generated with omega2, including the following command 
arguments: "-maxconfs 1 -strict false".
